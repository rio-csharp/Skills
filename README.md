# Skills

Personal Claude Code skills repository — reusable automations and workflows.

## Skills

- [siyuan](./siyuan/) — Interact with SiYuan (思源笔记) local API: search, read, create, update, delete notes, execute SQL queries, manage notebooks.

## Creating New Skills

This repository uses the [skill-creator](./skill-creator/) tool. Run `python skill-creator/scripts/scaffold_skill.py <name> --path .` to scaffold a new skill.
